# SEO Strategy for Smart Gift Finder

## Landing Page Template

Each landing page follows this structure:

### 1. Technical SEO
- Unique title tag (under 60 chars)
- Compelling meta description (under 160 chars)
- Canonical URL
- Schema markup (FAQPage, BreadcrumbList)
- Open Graph tags
- Twitter Cards
- Breadcrumb navigation

### 2. Content Structure
- H1: Main keyword (only one per page)
- H2: Section headers with related keywords
- H3: Sub-sections
- Table of Contents (improves UX and time on page)
- Internal links to related pages
- External links to authoritative sources (sparingly)

### 3. Conversion Optimization
- CTA above the fold
- Multiple CTAs throughout content
- "Not sure?" sections with AI gift finder link
- Related pages at bottom
- Social proof (coming soon)

### 4. Affiliate Link Strategy
- Prioritize high-commission links (featured badge)
- Provide 2-3 options per gift
- Use rel="nofollow noopener noreferrer"
- Track clicks with Google Analytics events
- Transparent affiliate disclosure

## Content Guidelines

### Word Count
- Landing pages: 1,500-2,500 words
- Blog posts: 2,000-3,000 words
- Product descriptions: 100-150 words

### Keyword Density
- Primary keyword: 1-2%
- LSI keywords: Natural placement
- Avoid keyword stuffing

### Internal Linking
- 3-5 internal links per page
- Use descriptive anchor text
- Link to related gift guides
- Link to AI gift finder

### External Linking
- Link to authoritative sources (sparingly)
- All affiliate links: nofollow
- Open in new tab
- Track all clicks

## Schema Markup Types

### FAQPage Schema
- Add 5-7 common questions
- Answer thoroughly (50-100 words each)
- Use natural language
- Target featured snippets

### BreadcrumbList Schema
- Show navigation hierarchy
- Improves UX and SEO
- Required on all pages

### Article Schema
- For blog posts
- Include author, date published, modified
- Image required

## Success Metrics

Track these KPIs weekly:
- Organic traffic growth
- Keyword rankings (track top 20 keywords)
- Bounce rate (target: under 60%)
- Time on page (target: over 2 minutes)
- Affiliate click-through rate
- Pages per session
