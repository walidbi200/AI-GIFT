# Pinterest Pin Images

**Size:** 1000 x 1500 pixels (2:3 ratio)
**Format:** JPG or PNG
**Purpose:** Pinterest sharing

## Design Guidelines:
- Text overlay with title
- Bright, eye-catching colors
- Large readable fonts (36pt+)
- Include site branding/logo
- High contrast

Create 5-10 pins per landing page with different designs.

Use Canva template "Pinterest Pin" (1000x1500)
